About The Blog

Hi, my name is Suraj. I created this blog site to put down ideas that I have and showcase my portfolio.

Credit

Here is what I use for building this site:

  • Zola: A fast static site generator in a single binary with everything built-in.
  • Pico: Minimal CSS Framework for semantic HTML
  • Feather Icon: Simply beautiful open source icons
  • JavaScript libraries:
    • Giscus: A comment system powered by GitHub Discussions.
    • Mermaid: JavaScript based diagramming and charting tool that renders Markdown-inspired text definitions to create and modify diagrams dynamically.
    • Katex: The fastest math typesetting library for the web.
    • MathJax: Beautiful and accessible math in all browsers
